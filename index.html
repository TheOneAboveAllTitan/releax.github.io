gnu freedom

Enjoy the output of Richard Stallmen free software movement, this gnu linux distro is free to modify , distribute , redistribute and do what the you want

fast and stable

releax gnu linux ships the stable version of linux kernel, and a light weight yet powerful xfce desktop environment , so that you use your ram not you os

customizable

Its linux, with xfce so, you can customize each and every thing you can imagine, or can or think . so that you have the system you want

independent

releax gnu linux is developed from scratch , and development is independent of any other linux distro, but inspired from a lots of other FOSS projects , like crux, arch , lfs

chatbot assistant

releax gnu linux ships with a very efficient chatbot assistant : nishu, that will help you to manage your system + you day to day task like compiling files, reminding things, search , play music and lots more,

powerful package manager

pkgs , a very powerfull package manager that help you to manager your packages, have capability to install many packages from other linux distro too.
